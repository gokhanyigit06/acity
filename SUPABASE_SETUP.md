# Supabase Setup Instructions

## 1. Environment Variables

Create a file named `.env.local` in the root directory and add your Supabase credentials:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

## 2. Database Schema

Run the following SQL in your Supabase SQL Editor to create the necessary tables for the homepage:

```sql
-- Create Brands table
create table brands (
  id bigint generated by default as identity primary key,
  name text not null,
  category text,
  logo_url text,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Insert some sample data
insert into brands (name, category) values
  ('Beymen Business', 'Giyim'),
  ('Network', 'Giyim'),
  ('Nike', 'Spor'),
  ('Adidas', 'Spor'),
  ('Mavi', 'Giyim'),
  ('Starbucks', 'Yeme İçme');
```
